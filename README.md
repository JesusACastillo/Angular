# Angular — espacio de aprendizaje

Repositorio reservado para prácticas y proyectos desarrollados con Angular. Actualmente no contiene código fuente publicable; esta documentación deja explícito el propósito y la ruta de crecimiento del proyecto.

## Estado

🚧 **En preparación.** La primera aplicación y su documentación técnica se añadirán en próximas versiones.

## Funcionalidades

Todavía no hay funcionalidades implementadas. El objetivo es reunir ejemplos progresivos de componentes, rutas, formularios, consumo de API y pruebas automatizadas.

## Tecnologías

Tecnologías previstas para la primera versión:

- Angular.
- TypeScript.
- HTML y CSS.
- RxJS.
- Jasmine/Karma o el stack de pruebas vigente al iniciar el proyecto.

## Instalación

No aplica todavía porque el repositorio no contiene una aplicación. Cuando se publique el proyecto, esta sección incluirá versiones requeridas y comandos reproducibles de instalación, desarrollo, pruebas y build.
